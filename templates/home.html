COMPUTER VISION PROJECT

1. FACE DETECTION

In this project I built a Face detector using OpenCV. There are various Haar cascades present on OpenCV's Github repository. OpenCV already contains many pre-trained classifiers for face, eyes, smiles, etc. For this project I used OpenCV's FrontalFace Haar Cascade Classifier which is a XML file. As per your need you can experiment with other classifiers as well. You can download the pre-trained classifier XML file haarcascade frontalface default.xml or you can visit OpenCV's Github repository.

2. OBJECT DETECTION

Object detection is probably the most profound aspect of computer vision due the number practical use cases. Object detection has been widely used for face detection, vehicle detection, security systems and driverless cars. Here I have created a project to detect objects using OpenCV and python. This Object Detection OpenCV method will be able to detect objects with good accuracy. For object detection from Webcam and Video I have used Mobile Net SSD object detector and for image used yolov3 object detector. Mobile Net SSD is faster than the yolov3 but gives less accuracy than yolov3.
